一图-精美的壁纸库.

这是公益性的壁纸库
我喜欢经常欢壁纸
有时候一张壁纸就能改变一个网站的风格
虽然壁纸可能有些不适合和自己的网站风格混搭
一个无聊的小作品
每个人都可以尽情的使用
现在已有2000+壁纸

如何使用？

壁纸地址 https://app.haotown.cn/img/bj@壁纸编号.jpg
你也可以使用JavaScript代码 可以替换ID为 oneimg的backgroundImage
//获取图片地址 function GetOneImg(){ OneImg="https://app.haotown.cn/img/bj@"+Math.round(Math.random()*2000)+".jpg" } //更换图片 function ChangeOneImg(){ console.log(OneImg);//调试图片地址 document.getElementById("oneimg").style.backgroundImage = "url("+OneImg+")"; GetOneImg(); LoadImg(OneImg);//预先加载下一张图片 } //预加载 function LoadImg(url) { var img = new Image(); img.src = url; }
当然 你也可以添加一个定时函数 实现自动换壁纸 别忘记了先要执行一下换壁纸操作
GetOneImg(); //获取随机图片地址 ChangeOneImg();//进行更换图片操作 setInterval(ChangeOneImg,8000);//重复更换图片
为了手机用户着想还可以把用以下语句判断以下
if(navigator.userAgent.match(/(iPhone|iPod|Android|ios|Mobile)/i)){ //手机用户执行代码区域 } else { setInterval(ChangeOneImg,8000);//非手机用户重复更换图片 }
